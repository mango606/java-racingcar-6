# 자동차 경주 기능 목록

## Application 클래스
- 게임 시작 및 사용자 입력 처리
- Car 객체들을 초기화하고 리스트로 반환
- 모든 자동차에 대해 랜덤 값 생성 후, 조건에 맞으면 전진 처리
- 모든 자동차의 현재 위치 출력
- 우승자 찾기
- 최대 위치 값 찾기

## Car 클래스
- Car 객체 생성 시 이름 초기화
- 자동차 전진 처리 (위치 1 증가)
- 자동차 이름 반환
- 자동차 현재 위치 반환

## 유틸리티
- 지정된 범위 내에서 랜덤 숫자 생성

## 인터페이스
자동차가 전진할 조건을 제공 (예: 랜덤 숫자가 4 이상일 경우 전진)

## 예외 처리
- 자동차 이름 검증 실패 시 발생
- 사용자 입력 검증 실패 시 발생
